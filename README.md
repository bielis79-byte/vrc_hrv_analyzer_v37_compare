# VRC / HRV RRi Analyzer v3.7

Aplicación Streamlit para análisis y comparación de variabilidad del ritmo cardíaco.

Incluye:

- Basal + E1-E6 + R1-R3.
- Ventanas de duración libre.
- Comparación real entre dos o más registros.
- Gráficas comparativas por variable y fase.
- RRi superpuesto por fase.
- HRV lineal, frecuencial, no lineal, RQA, Poincaré y HVG.
- Exportación Excel y ZIP.

Ejecutar:

```bash
streamlit run app.py
```
